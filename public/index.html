Sobre o Projeto

O Football News é um portal de notícias esportivas dedicado ao futebol nacional e internacional. O site apresenta as principais notícias, resultados e análises dos campeonatos mais importantes do mundo, com atualizações sobre times brasileiros e europeus.

O projeto foi desenvolvido como trabalho prático da disciplina, utilizando Bootstrap 5, JavaScript puro e uma estrutura de dados JSON para renderizar o conteúdo de forma dinâmica.
